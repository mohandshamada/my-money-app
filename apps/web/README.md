# CashFlow Web Frontend

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- **Landing Page** - Marketing, pricing, features
- **Auth** - Login, Register with JWT
- **Dashboard** - Overview with stats and recent transactions
- **Transactions** - List, add, import CSV
- **Budgets** - Create and manage budgets
- **Forecast** - Cash flow projection with charts

## Tech Stack

- Vite + React + TypeScript
- TailwindCSS
- Redux Toolkit
- React Router
- React Query
- Recharts
